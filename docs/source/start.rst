Getting Started
===============

**nnsight** (/ɛn.saɪt/) is a package for the interpreting and manipulating the internals of deep learning models.

.. _installation:

Installation
------------

To get started with nnsight, install it with ``pip``. 

.. code-block:: console

   pip install nnsight
