nnsight Source Serialization Documentation

Documentation for nnsight's source-based serialization system, which enables Python-version-independent remote execution on NDIF servers.

Tutorials

  • Source Serialization Tutorial

    Comprehensive guide to how source-based serialization works, including the @remote decorator, serialization format, and special case handling.

Design Documents

  • Source Serialization Design

    Main design document covering the architecture, wire format, and implementation details of source-based serialization.

  • Forbidden Serialization Design

    Design for early detection and blocking of objects that should not be serialized (pandas DataFrames, sockets, pytest fixtures, etc.) with actionable error messages.

  • LocalSimulationBackend Design

    Design for the remote='local' backend that simulates NDIF serialization locally for testing and debugging.

  • Version Check Optimization

    Future optimization to skip source transmission when the server has a compatible library version installed.

Development Tools

  • Call Graph Visualization

    Interactive visualization of function call relationships in the serialization codebase.

  • Data Flow Visualization

    Interactive visualization of data flow through the serialization pipeline.
